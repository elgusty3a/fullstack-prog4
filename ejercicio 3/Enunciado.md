# Ejercicio 3 — Interface + Herencia

## Crear una interface llamada Reproducible con métodos relacionados con la reproducción de contenido.

Luego crear una clase base llamada Contenido que represente contenido multimedia.

Los atributos quedan a elección.

Después crear dos clases:

-   Video
-   Podcast

Estas clases deben utilizar herencia y la interfaz definida anteriormente.

Cada clase debe implementar su propia lógica para los métodos de reproducción.

Finalmente, crear instancias y probar su comportamiento.
