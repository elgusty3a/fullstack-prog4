# Ejercicio 1 — Interfaces

## Crear una interface llamada Notificable.

Esta interfaz debe definir métodos relacionados con el envío de notificaciones.

Luego crear dos clases que implementen esta interfaz:

-   Email
-   SMS

Cada clase debe implementar los métodos definidos en la interfaz.

Los atributos de las clases quedan a elección, siempre que tengan sentido con el tipo de notificación.

Finalmente, crear instancias de ambas clases y probar el envío de mensajes.

### Para compilar
```
npm run build
```
### Para ejecutar
```
npm start
```
