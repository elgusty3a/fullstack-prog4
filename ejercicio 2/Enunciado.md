# Ejercicio 2 — Herencia

## Crear una clase base llamada Cuenta que represente una cuenta dentro de un sistema financiero.

La clase debe incluir métodos relacionados con operaciones típicas de una cuenta, como por ejemplo depositar o retirar dinero.

Los atributos quedan a elección.

Luego crear dos clases que extiendan de Cuenta:

-   CuentaAhorro
-   CuentaCorriente

Cada clase puede agregar comportamiento propio o redefinir alguno de los métodos heredados.

Finalmente, crear instancias de ambas clases y probar sus métodos.

### Para compilar
```
npm run build
```
### Para ejecutar
```
npm start
```
